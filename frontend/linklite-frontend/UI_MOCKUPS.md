## 📱 Responsive Behavior

### Mobile (< 640px)
```
Full width layout:
- Header: Stacked title and nav
- Forms: Full width inputs
- Cards: 1 column grid
- Stats: Stacked vertically
- No "API Connected" badge
```

### Tablet (640px - 1024px)
```
Medium layout:
- Header: Horizontal with nav
- Forms: 80% width
- Cards: 2 column grid
- Stats: Full width bar
```

### Desktop (> 1024px)
```
Full layout:
- Header: Full horizontal menu
- Forms: 50% width
- Cards: 2 column grid
- Stats: 3 column grid
- API badge visible
- Full feature cards display
```

## 🖱️ Interaction Patterns

### Copy Click Pattern
```
User clicks [Copy]
     ↓
Copies URL to clipboard
     ↓
Button icon changes to ✓
     ↓
Color changes to green
     ↓
2 seconds pass
     ↓
Icon/color reverts to [Copy]
```

### Delete Pattern
```
User clicks [🗑️]
     ↓
Modal appears: "Are you sure?"
     ↓
Options: [Cancel] [Delete]
     ↓
If Delete:
  ├─ API request sent
  ├─ Card removed from UI
  ├─ Stats updated
  └─ Success message shown
```

### View Details Pattern
```
User clicks [View Details]
     ↓
Analytics detail page loads
     ↓
Shows:
  ├─ Stats cards
  ├─ URL info
  ├─ Charts
  └─ Click history
     ↓
Back arrow available
```
